Simnor Widgets a selection of widgets to use on your site including twitter, dribbble, latest posts, social links and author list.

== Description ==

Simnor Widgets a selection of widgets to use on your site including:

Twitter feed widget: shows your latest tweets, this uses the OAuth authentication through the Twitter 1.1 API, so you'll need to get api keys, a link to a simple tutorial can be found when adding the widget.

Dribbble shots widget: shows the latest shots from a dribbble account.

Latest posts widget: lists the latest posts with the thumbnail image.

Social links widget: shows a selection of social icons on your site.

Authors widget: list of authors on your blog including a post count for each author.

== Installation ==

1. Upload the `simnor-widgets` folder to your to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the Appearance -> Widgets page and start adding your widgets.
